Controls

Moving

  • D - Move positively along the X axis
  • A - Move negatively along the X axis
  • W - Move positively along the Y axis
  • S - Move negatively along the Y axis
  • Q - Move positively along the Z axis
  • E - Move negatively along the Z axis

Rotating

  • H - Rotate positively along the X axis
  • F - Rotate negatively along the X axis
  • T - Rotate positively along the Y axis
  • G - Rotate negatively along the Y axis
  • R - Rotate positively along the Z axis
  • Y - Rotate negatively along the Z axis

Scaling

  • L - Scale positively along the X axis
  • J - Scale negatively along the X axis
  • I - Scale positively along the Y axis
  • K - Scale negatively along the Y axis
  • U - Scale positively along the Z axis
  • O - Scale negatively along the Z axis
  • X - Increase scale uniformly
  • Z - Decrease scale uniformly

Clear all transforms

  • C - Reset all transforms on selected object
